# Color Detection Project

This project is a color detection application built with Python, Flask, OpenCV, and pandas. It lets users upload an image in the browser, click any area, and view the matched color from `color.csv`. The project also includes a training and testing notebook and a legacy OpenCV desktop version.


## Dataset Source

The dataset used in this project was taken from Kaggle:

```text
https://www.kaggle.com/
```

## Run This Project

1. Open terminal in this folder:

```text
C:\Users\lalan\OneDrive\Desktop\Color-Detection
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the project:

```bash
python app.py
```

## Local Notebook And Desktop Dependencies

If you also want the notebook and the legacy OpenCV desktop script, install:

```bash
pip install -r requirements-dev.txt
```
